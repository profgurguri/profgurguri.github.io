O número é um conceito abstrato que representa a ideia de quantidade.

Um sistema de numeração é um conjunto de símbolos (um alfabeto) utilizado para representa quantidades e regras que definem a forma de representação.

Vamos supor que existe uma sistema de numeração chamado baseX.

Um sistema de numeração efetivamente útil deve:

  1. Ser capaz de representar uma grande quantidade de números.
