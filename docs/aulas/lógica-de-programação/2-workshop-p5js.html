Em vez de procurar de forma inteligente, você decide pegar cada chave uma por uma e tentar abrir a porta. Essa é a ideia básica de um algoritmo de força bruta.

Vamos fazer um “break”
...

O break é usado para sair imediatamente de um laço. É como dizer: "Pare tudo por aqui e vá para a próxima parte do código."

JavaScript
1
for (let i = 0; i < 10; i++) {
2
  if (i === 5) {
3
    break; // Quando i for 5, o laço será interrompido
4
  }
5
  console.log(i);
6
}

Já foi, vamos “continue”
...

O continue pula para a próxima iteração do laço. É como dizer: "Ignore o resto desta iteração e vá para a próxima."

JavaScript
1
for (let i = 0; i < 10; i++) {
2
  if (i % 2 === 0) {
3
    continue; // Pula para a próxima iteração se i for par
4
  }
5
  console.log(i);
6
}
