Convenções Comuns em JavaScript:
...

  • Camel Case:
    • A primeira letra de cada palavra, exceto a primeira, é maiúscula.
    • É a convenção mais comum em JavaScript.
    • Exemplos: firstName, lastName, productPrice.
  • Snake Case:
    • Palavras são separadas por underscores (_).
    • Exemplos: first_name, last_name, product_price.
  • Pascal Case:
    • A primeira letra de cada palavra é maiúscula.
    • Exemplos: ProductName, CustomerInfo.

Cores em p5.js
...

Esquema de representação de cores
...

  • Codificação de Escala de Cinza
    • número inteiro no intervalo
    • 0 “preto”, 255 “branco”;

Codificação de RGB (Red, Green, Blue — Vermelho, Verde, Azul)
...

  • conjunto de 3 (três) inteiros no intervalo
  • (255,0,255) “misture o máximo de vermelho com o máximo de azul”

Roxo

  • Codificação RGB + alpha (opacidade)
    • alpha: é um número no intervalor
    • 0 “transparente”
    • 255 “opaco”
  • ⚠ Nome da cor em english: “red”, “blue”, “indigo”, “olive”
  • ⚠ Cores em Hexadecimal: “#FF0000”

Operadores Relacionais
...

Operador Descrição Exemplo
== Igualdade (comparação de valores) 3 == "3" (retorna true, pois os valores são iguais)
=== Igualdade estrita (compara valor e tipo) 3 === "3" (retorna false, pois os tipos são diferentes)
!= Desigualdade (comparação de valores) 3 != 4 (retorna true, pois os valores são diferentes)
!== Desigualdade estrita (compara valor e tipo) 3 !== "3" (retorna true, pois os valores e tipos são diferentes)
> Maior que 5 > 3 (retorna true)
< Menor que 2 < 4 (retorna true)
>= Maior ou igual a 5 >= 5 (retorna true)
<= Menor ou igual a 3 <= 4 (retorna true)
